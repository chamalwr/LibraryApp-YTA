# Library API - NodeJS + Express

## Before Running

Create .env file to the sctructure given below

```bash
# DB Configs
MONGODB_ATLAS_URL="<CONECTION STRING>" #This used as the default connection string
MONGODB_LOCAL_URL=""

# App Configs
PORT=<PORT_NUMBER>

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
